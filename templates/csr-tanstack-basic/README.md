# Coral - An experimental DX first full stack framework

## CSR - TanStack Router Basic Starter App

A minimal **client-side rendered (CSR)** starter built with **TanStack Router**, designed for fast iteration and a clean developer experience.

---

## Overview

This starter provides a lightweight foundation for building single-page applications using:

* File-based routing for client
* Fully client-side navigation
* Minimal configuration
* Basic API route support

It is intended as a **baseline template**—you can extend it with state management, data fetching, or backend integration as needed.

---

## Features

* Client-side rendering (CSR)
* Routing with TanStack Router
* Component-driven structure
* Minimal and unopinionated setup
* Easy to extend

---

## Getting Started

### Install via CLI

```
 npx create-coral-app
```

### Start development server

```bash
npm run dev
```
